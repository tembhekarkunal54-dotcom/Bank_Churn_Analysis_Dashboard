#  Bank Churn Analysis Dashboard

##  Overview
This Power BI dashboard analyzes customer churn patterns for a bank. It helps identify key factors that contribute to customer attrition and provides actionable insights for retention strategies.

##  Key Metrics
- **Total Customers:** 10,000+
- **Churned Customers:** 2,037
- **Churn Rate:** 20.37%
- **Average Credit Score:** 650.53

## Dashboard Features
- **Customer Demographics:** Distribution by Country, Gender, Age Group
- **Churn Analysis:** Churn rate by age, country, and product usage
- **Financial Insights:** Credit card holders vs non-holders
- **Interactive Filters:** Dynamic exploration of data

##  Visuals Included
1. KPI Cards - Total Customers, Lost Customers, Churn Rate, Avg Credit Score
2. Customers by Country - Geographic distribution
3. Churn vs Non-Churn - Donut chart
4. Age-wise Churn Analysis - Bar chart
5. Country & Gender Distribution - Stacked bar chart
6. Credit Card Holders - Card ownership analysis
7. Product Usage by Gender - Product analysis

##  Tools Used
- **Power BI Desktop**
- **DAX Measures** for calculations
- **CSV Dataset** (Bank customer data)

##  Files
- `Bank_Churn_Analysis_Dashboard.pbix` - Main Power BI dashboard

##  How to Use
1. Download the `.pbix` file from this repository
2. Open with Power BI Desktop
3. Explore interactive visualizations
4. Apply filters to analyze specific customer segments
5. Gain insights for customer retention strategies

##  Key Insights
- Churn rate is highest in certain age groups
- Country-wise variations in customer retention
- Credit card ownership impacts churn behavior
- Product preferences differ by gender
